# Productivity Timer Useful v2

A productivity tool designed to promote healthier use of social media by helping users stay on task and purposeful. The application uses a timer and a checklist to manage activities within a given time block, ultimately blocking access to specified websites once the time is up.

## Features

- **Timer**: Set time blocks for focused social media use.
- **Task Checklist**: Create and manage a list of tasks to complete within each time block.
- **Website Blocking**: Automatically block specified websites when the timer expires to encourage healthier usage habits.
- **User-Friendly Interface**: Simple and intuitive design for ease of use.

## Installation

To install and run the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/mattySIguy/website_timer_and_task_list.git
   ```

2. Navigate to the project directory:

   ```bash
   cd productivity_timer_useful_v2
   ```

3. Install the necessary dependencies:

   ```bash
   npm install
   ```

## Usage

To start using the productivity timer and task list, run the following command:
```bash
npm start
```

Once the application is running, you can set timers and manage your tasks through the user interface. The application will block specified websites once the timer expires, helping you maintain focus and use social media more purposefully.
